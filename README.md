## Implementation
+ ```Heroku```, to deploy the app
+ FrontEnd testing with ```Jasmin```
+ Backend testing with ```tape```
+ ```Istanbul``` - at least 90% coverage on backend
+ ```SASS``` for the ```CSS```
+ ```HAPI``` framework
